###接口
####获取用户是否关注公共号
接口调用请求说明
> http请求方式: GET https://xxxx.xx.com?openid=OPENID

参数说明

参数|是否必须|说明
:----------- | :----------- | :-----------:
openid|是|普通用户的标识，对当前公众号唯一
返回说明
正常情况下，返回下述JSON数据包：
```
{
   "subscribe": 1
}
```
参数说明

| 参数 |说明 |
| --- | --- |
|subscribe |用户是否订阅该公众号标识，值为0时，代表此用户没有关注该公众号  |
